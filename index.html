About Car Racing

A heartfelt racing adventure built with family, creativity, and powerful HTML/CSS/JavaScript.

Family built · Made for nieces

Welcome to Car Racing! 🏁

This game is a personal project created with HTML, CSS & JavaScript. It was made to provide a fun and simple racing experience for everyone.

This game is a thrilling, fast-lane racing adventure featuring diverse environments like City, Desert, and Snow, designed to test your high-score skills.

❤️ A Personal Message

I created this game with love for my little nieces (my sister's daughters), so they can enjoy playing something made by their own family.

This project also has a special place in my heart. My father is no longer with us, and I built this game as a way to improve my skills and make him proud. Even though he isn't here today, every line of code reminds me to keep learning and moving forward. I really missing a lot Papa & Love you ❤️‍🩹🥹
